
### 使用：

1. `npm install` 或 `yarn`

2. 打开浏览器的开发者工具, 选中具体的请求, 右键, Copy  => Copy as Node.js fetch, 复制到index.js

3. windows设置定时器, 在powershell中运行schtasks中的命令
